# Stegano Image (AES-256 Encrypted Steganography Tool)

画像に秘密のメッセージを埋め込むことができる、セキュリティ特化型のステガノグラフィツール。
見た目を変えずにデータを隠蔽するLSB手法に加え、Web Crypto APIを用いたAES-GCM 256bit暗号化を実装し、高い機密性を実現した。

**デモサイト:** https://yh-27182.github.io/Stegano_Image/

![スクリーンショット](images/screenshot.png)

## 特徴

* **二重のセキュリティ:**
    * **隠蔽 (Steganography):** 画像の最下位ビットを操作し、肉眼では判別できないようにデータを埋め込む
    * **暗号化 (Cryptography):** AES-GCMアルゴリズムを使用することで、第三者がデータを抽出しても復号不可能
* **サーバーレスで安全:**
    * 全ての処理はブラウザ上で完結し、画像やパスワードがサーバーに送信されることはないため、プライバシーが完全に守られる

## 技術スタック

* **Languages:** HTML5, CSS3, JavaScript
* **Core APIs:**
    * Web Crypto API (AES-GCM, PBKDF2) - 暗号化処理
    * Canvas API (ImageData) - ピクセル操作

## 技術的なこだわり

### 1. AES-GCMによる強力な暗号化
単なるXOR演算などの簡易的な難読化ではなく、AES-GCM (256bit) を採用した。

* **鍵導出:** ユーザーのパスワードから直接鍵を作るのではなく、PBKDF2を用いてソルト(Salt)と共に10万回のストレッチングを行い、ブルートフォース攻撃への耐性を高めている。
* **ランダム性:** 暗号化のたびにランダムな初期化ベクトルとSaltを生成することで、同じ画像・同じパスワード・同じ文章であっても、毎回異なるバイナリデータが生成される。

### 2. LSB法の実装
画像データの各ピクセルの最下位ビットを、暗号化データのビットに置き換えるアルゴリズムを自作した。
これにより、画質の変化を1/256に抑え、人間の目には全く変化がないように見える。
